# DuoGrid
пошаговая стратегия на двоих, сделанная на Unity, главное отличие которой битва на изменяемых полях с катастрофами, мутациями и всяким разным дополнениями. За разных персонажей, с разными таллантами и механиками.
Каждый игрок управляет магом, который может менять клетки на полях.
Главная фишка - финиш ставит тоже игрок, но финиш максимум только один на всю карту.

## Всякие механики
- **Огонь**: это клетка которая может убрать любую не защищенную растительность.
- **Грибная катастрофа**: Событие при котором появляются грибы, с возможностью смутировать.
- **Грибы, Катусы**: по факту одна из разрастающихся механик на момент написания. Выживания, Мутации, Тактики.
- **Таланты**: Возможность выбора усиления после каждого раунда.

## Сама игра
Игра проходит до момента победы игрока n-ое кол-во раз. Выбираемое в настройках игры.
1. При запуске стандартной игры, каждый игрок выбирает себе Персонажа.
2. После выбора выбираются начальные клетки и предметы игроков.
3. Выбор талантов. Таланты могут выпасть рандомные до определённых уровней на каждого героя.

## Близжайшие дополнения

Виды магов, Выборы магов
